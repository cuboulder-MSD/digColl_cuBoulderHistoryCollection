# University of Colorado Boulder History Collection
## Description
The University of Colorado Boulder History Collection contains university records from the University Libraries' archives that document the university's history and evolution from 1876 to the present.

Permanent address to digital collection: [https://doi.org/10.60807/4zq9-pj92](https://doi.org/10.60807/4zq9-pj92).<br /> 
Please use the same link when sharing or citing the collection.
